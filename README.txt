Robert Redford Fan Site
---------------------

 * By Jordan Navratil and Jenna Navratil, October 12, 2015
 * A simple Robert Redford fansite built using Drupal
 

Setup
------------
 * Clone repository
 * Import the database found in sites/db-backup to MySQL
 * Database Credentials:
     username: admin
     password: admin
 * Start Apache Server
 * Browse to localhost:8888
 * Site Admin Login:
     username: admin
     password: admin

Technologies Used
--------------------------
 * Drupal 7.40
